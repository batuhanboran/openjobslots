# SEO Audit Runbook

This runbook is for public `openjobslots.com` crawlability fixes. It covers the static web/SEO surface only; it must not be used as approval for ATS source writes, production backfills, Meilisearch replace reindexes, or worker changes.

## Semrush May 31, 2026 Targets

- Site health target: `98-100%`.
- Errors target: `0`.
- Low word count target: `0` public HTML landing pages under 200 meaningful words.
- Hreflang language mismatch target: `0`.
- Orphaned sitemap page target: `0`.

## App-Owned SEO Surface

- Route catalog: `src/publicSeoRoutes.js`.
- HTML metadata, fallback content, robots, sitemap, and JSON-LD rendering: `server/http/publicSeo.js`.
- Route registration and cache headers: `server/http/registerPublicRoutes.js`.
- Main verification command: `npm.cmd run test:http`.

The app-owned `robots.txt` must stay standards-compatible:

```txt
User-agent: *
Allow: /
Disallow: /applications
Disallow: /settings
Disallow: /sync
Disallow: /ingestion
Disallow: /mcp
Disallow: /frontend
Disallow: /postings
Sitemap: https://openjobslots.com/sitemap.xml
```

Do not add non-standard directives such as `Content-Signal` to the app-generated robots output.

## Cloudflare Robots Injection

The May 31, 2026 live audit showed Cloudflare adding a managed robots block that included:

```txt
Content-Signal: search=yes,ai-train=no
```

Semrush reported that line as `Invalid robots.txt format`. This is not generated by the repository. Fix it in Cloudflare by disabling the managed robots content-signal injection, or by configuring the zone so AI/content-signal policy is expressed outside the standards-compatible robots file.

After any Cloudflare change, verify:

```powershell
curl.exe -fsS https://openjobslots.com/robots.txt
```

The live file should not contain `Content-Signal`, should keep the sitemap line, and should not include `Disallow: /` for `User-agent: *`.

## Crawl Verification

After SEO code changes and deploy:

```powershell
npm.cmd run test:http
npm.cmd run build:web
curl.exe -I https://openjobslots.com/
curl.exe -fsS https://openjobslots.com/robots.txt
curl.exe -fsS https://openjobslots.com/sitemap.xml
```

Then rerun Semrush Site Audit and compare:

- `Invalid robots.txt format`
- `Low word count`
- `Hreflang language mismatch issues`
- `Orphaned sitemap pages`
